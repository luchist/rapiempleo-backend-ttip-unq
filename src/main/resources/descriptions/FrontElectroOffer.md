#PotenciáTuDesarrolloProfesional

Queremos sumar dos desarrolladores FrontEnd para nuestra software factory interna, que participarán de una célula orientada a un 
nuevo producto dentro de la unidad de negocios de servicios transaccionales.

**Principales funciones:**

- Desarrolla y mantiene interfaces y funcionalidades web orientadas al registro de usuarios, gestión de cuentas, participación 
en juegos lotéricos y visualización de resultados, con el objetivo de habilitar un canal digital confiable y fácil de usar 
para los apostadores. 
- Implementa la integración de las aplicaciones con servicios y APIs de la plataforma que soportan las 
operaciones transaccionales, con el fin de asegurar la correcta ejecución de las apuestas y la gestión de usuarios. 
- Analiza y optimiza la experiencia de usuario, la accesibilidad y el rendimiento de las aplicaciones, para garantizar una 
navegación ágil, estable y consistente. 
- Aplica buenas prácticas de desarrollo, revisiones de código, testing y documentación técnica, con el propósito de asegurar 
la calidad, mantenibilidad y escalabilidad del software. 
- Asimismo, colabora con equipos de producto, diseño, backend y calidad en la planificación e implementación de nuevas funcionalidades, 
para contribuir a la evolución continua del canal digital.


**Requisitos/Requerimientos:**

- 5+ años de experiencia en desarrollo con Next.js con TypeScript
- 5+ años de experiencia en desarrollo con HTML5 y CSS/SCSS
- 5+ años de experiencia en desarrollo e integrando con Strapi Headless CMS
- 5+ años de experiencia con sistema de versionado GIT
- Alto nivel de conocimiento con consumo de APIs Restful
- Alto nivel de conocimiento desarrollando sitios Responsivos (Mobile First)
- Conocimientos básicos de Experiencia de Usuario (UX)
- Conocimientos básicos de Figma
- Excelentes habilidades de comunicación
